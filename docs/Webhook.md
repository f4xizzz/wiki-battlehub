# **Webhook**

---

## **Discord Webhook Integration**

The **Cobblemon BattleHUB** comes equipped with a real-time communication bridge to Discord. The mod can notify your community about tournament progress, match calls, results, and champion coronations.

It also automatically generates and attaches the live bracket visualization.

---

### **File Path**

`config/cobblemon_battlehub/webhook.json`

---

## **Dynamic Variables and Placeholders**

The mod dynamically replaces terms enclosed in curly braces {} before sending the payload to Discord. You can use these variables in the content, title, or description of any message:

| Placeholder | What it replaces |
| :---- | :---- |
| {name} | Tournament display name defined in the profile.
| {participants} | Number of players registered for the competition.
| {phase} | Identifier of the current match block/phase.
| {p1} | Name of the first player called for the match.
| {p2} | Name of the second player called for the match.
| {winner} | Name of the player who won the match or tournament.
| {loser} | Name of the player who lost and was eliminated.
| {bracket} | Dynamic link or status text for the bracket.
