# **Chat Config**

---

## **Step-by-Step Setup**

1. **Ensure Installation:** Make sure **CarbonChat** is installed and functioning correctly on your Fabric environment.  
2. **Sync the IDs:** Open the CarbonChat channel configuration file and verify your global and local channel IDs. Replace them in `globalChannelId` and `localChannelId` inside `chatconfig.json`.  
3. **Customize the Appearance:** Modify `tabGlobalName` and `tabLocalName` if you want them to display styled names like [GLOBAL] or [ARENA].  
4. **Reload on the Server:** To apply chat changes in real time without restarting, run the command:  
   `/bh reload`

---

### **File Path**

`config/cobblemon_battlehub/chatconfig.json`

!!! info "Chat Mod Compatibility"
    Currently, **Cobblemon BattleHUB** has native and exclusive support for the **CarbonChat** mod.

    If your server uses another chat mod or management system (such as *EssentialCmds*, *LPC*, etc.), don’t worry! You can join our [Official Discord](https://discord.gg/aDCgBbvRe5) and open a support ticket. We will happily provide a custom integration for your server's chat system.

## **Default Configuration Template**

Below is the default template generated by the `ChatConfig` class on first startup:

        {  
         "globalChannelId": "global",  
         "localChannelId": "local",  
         "globalCommand": "g",  
         "localCommand": "l",  
         "tabGlobalName": "GLOBAL",  
         "tabLocalName": "LOCAL"  
    }

---

## **Detailed Parameter Explanation**

### **1. Server Identifiers (CarbonChat)**

These parameters must match the channel IDs configured inside CarbonChat for BattleHUB communication to work properly.

* **globalChannelId** (Default: "global"): The internal ID of the global chat channel used by CarbonChat on the server.  
* **localChannelId** (Default: "local"): The internal ID of the local chat channel used by CarbonChat (usually for specific arena chats).

### **2. Client-Side Commands**

These are the shortcuts players use in chat to quickly switch between the integrated chat tabs and channels.

* **globalCommand** (Default: "g"): The quick command typed in chat to switch to the global channel (e.g. `/g <message>`).  
* **localCommand** (Default: "l"): The quick command typed in chat to switch to the local arena channel (e.g. `/l <message>`).

### **3. Tab Visual Customization (GUI)**

These values change the appearance of the tabs shown to players in the mod's own chat interface (battle HUD).

* **tabGlobalName** (Default: "GLOBAL"): The display name players will see on the global channel tab in the mod interface.  
* **tabLocalName** (Default: "LOCAL"): The display name players will see on the local channel tab in the mod interface.
