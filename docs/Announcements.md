# **Announcements Configuration**

The announcement system in **Cobblemon BattleHUB** allows you to display news, important notices, and links directly in the mod menus (GUI) for players. It uses the modern **MiniMessage** text format, supporting colors, gradients, and even clickable links that open a browser.

---

### **File Path**

`config/cobblemon_battlehub/announcements.json`

## **Default Configuration Template**

Below is the official template automatically generated by the AnnouncementsManager class:

        {  
        "version": 1,  
         "messages": [  
          "<gradient:gold:yellow><bold>[NEWS]</bold></gradient> <white> Season 1 has started!<white>",  
         "<click:open_url:'https://discord.gg/aDCgBbvRe5'><aqua><u>Join our Discord!</u></aqua></click>"  
         ]  
        }

---

## **Detailed Parameter Explanation**

### **1. Notification Control (version)**

* **version** (Default: 1):  
  This integer acts as the player's pending announcement tracker.  
  Whenever a player joins the server, the client compares this number with their local file (`announcements_state.txt`). If you update `announcements.json` and **increase the version number** (for example, from 1 to 2), the game will detect new content and show an **Unread / New** visual alert to all players.

### **2. Visual Content (messages)**

* **messages** (String Array):  
  A list of announcement lines to display in the player's interface. You can add as many lines as you want using the rich formatting features of **MiniMessage (Kyori)**.

---

## **How to Style and Create Great Announcements**

The mod interprets **MiniMessage** syntax, so you do not need to rely on legacy Minecraft color codes like `&a` or `&c`. Here are some supported formatting options:

### **1. Colors and Gradients**

* **Simple Colors:** `Use <red>, <blue>, <green>, <gold>, etc.`
    * *Example:* `<green>New update available!</green>`
* **Gradients:** `Create smooth color transitions using <gradient:color1:color2>.` 
     * *Example:* `<gradient:gold:yellow><bold>[UPDATE]</bold></gradient>`

### **2. Text Formatting**

* **Bold:** `<bold> or <b>`
* **Underline:** `<underline> or <u>`
* **Italic:** `<italic> or <i>`

### **3. Clickable Links (Click Events)**

You can make text open the player's browser when clicked:

* **Link Syntax:** `<click:open_url:'LINK'>Clickable Text</click>`
    * *Practical Example:* `<click:open_url:'https://discord.gg/aDCgBbvRe5'><aqua><u>Click here to join Discord!</u></aqua></click>`

---

## **How to Reload New Messages**

After changing the messages or increasing the version in `announcements.json`, apply the update immediately on the server and sync the UI for all online players by running:

`/bh reload`
